Features

- Ingestion Pipeline
    - Pipeline Orchestration: Airflow
    - Storage: minIO & S3
    - VectorDB: qdrant & milvus
    - GraphDB: neo4J
    - Feature Store: feats
    - Embedding: OpenAI compatible (vLLM hosting)
    - LLM: OpenAI compatible (third-party APIs)

- Retrieval Pipeline
    - Embedding: OpenAI compatible (vLLM hosting)
    - LLM: OpenAI compatible (third-party APIs)
    - Observability: Langfuse

- Synthetic data generation Pipeline
    - DeepEval

- Evaluation Metrics
    - DeepEval
    - Ragas

- UI: NotebookLM
